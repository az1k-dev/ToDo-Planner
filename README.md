# ToDo-Planner
📝📅 Telegram planner bot


## In the bot, you can:
+ 📄 Watch today's tasks
+ 💠 Add new task
+ 📝 Edit tasks
    + ✏️ Edit task's title
    + 📆 Edit task's days of the week
    + 🗑 Delete task
+ 🚩 Select a language
